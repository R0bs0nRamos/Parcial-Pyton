Explicación del Código: Programa para Contar Años Bisiestos

Este programa solicita dos años al usuario y calcula cuántos años bisiestos hay en ese intervalo.

Visión General del Programa

El programa está diseñado para:

  1. Solicitar al usuario que ingrese dos años.
  2. Validar las entradas para asegurarse de que sean números enteros y que el primer año no sea mayor que el segundo.
  3. Calcular cuántos años bisiestos existen entre los dos años (inclusive).
  4. Mostrar la cantidad y la lista de los años bisiestos encontrados.
  5. Ofrecer al usuario la opción de realizar otra consulta o finalizar el programa.

Ahora, vamos a detallar cada parte del código.

1. Función es_bisiesto(anio)

def es_bisiesto(anio):
    """Determina si un año es bisiesto."""
    return (anio % 4 == 0) and (anio % 100 != 0 or anio % 400 == 0)

Propósito: Esta función verifica si un año determinado es bisiesto.

Cómo Funciona:

  • Condición Principal: Un año es bisiesto si es divisible por 4 y (no es divisible por 100 o es divisible por 400).
  • Expresión Booleana:
    • anio % 4 == 0: Verifica si el año es divisible por 4.
    • anio % 100 != 0: Verifica si el año no es divisible por 100.
    • anio % 400 == 0: Verifica si el año es divisible por 400.
  • Retorno: La función devuelve True si el año es bisiesto y False en caso contrario.

Ejemplos:

  • 2000:
    • Divisible por 4? Sí.
    • Divisible por 100? Sí.
    • Divisible por 400? Sí.
    • Resultado: Bisiesto.
  • 1900:
    • Divisible por 4? Sí.
    • Divisible por 100? Sí.
    • Divisible por 400? No.
    • Resultado: No es bisiesto.
  • 2016:
    • Divisible por 4? Sí.
    • Divisible por 100? No.
    • Resultado: Bisiesto.

2. Función obtener_años_bisiestos(inicio, fin)

def obtener_años_bisiestos(inicio, fin):
    """Retorna una lista de años bisiestos entre inicio y fin (inclusive)."""
    return [anio for anio in range(inicio, fin + 1) if es_bisiesto(anio)]

Propósito: Esta función genera una lista de todos los años bisiestos entre dos años proporcionados (incluyendo los años de inicio y fin).

Cómo Funciona:

  • Comprensión de Lista:
    • range(inicio, fin + 1): Crea una secuencia de años desde inicio hasta fin, incluyendo fin (por eso el +1).
    • for anio in range(...): Itera sobre cada año en la secuencia.
    • if es_bisiesto(anio): Verifica si el año actual es bisiesto utilizando la función es_bisiesto.
    • Si la condición es verdadera, el año se incluye en la lista resultante.
  • Retorno: La función devuelve una lista que contiene todos los años bisiestos en el intervalo especificado.

Ejemplo: Para inicio = 2000 y fin = 2020, la lista retornada sería [2000, 2004, 2008, 2012, 2016, 2020].

3. Función solicitar_años()

def solicitar_años():
    """Solicita al usuario ingresar dos años y los valida."""
    while True:
        try:
            anio1 = int(input("Ingrese el primer año: "))
            anio2 = int(input("Ingrese el segundo año: "))
            if anio1 > anio2:
                print("Error: El primer año debe ser menor o igual al segundo año. Inténtelo de nuevo.")
                continue
            return anio1, anio2
        except ValueError:
            print("Error: Por favor, ingrese un número válido para los años.")

Propósito: Solicita al usuario ingresar dos años y valida que las entradas sean correctas.

Cómo Funciona:

  • Bucle Infinito (while True): Permite que el usuario intente ingresar los años nuevamente en caso de error.
  • Bloque try:
    • Intenta convertir las entradas del usuario a enteros utilizando int(input(...)).
  • Validación de las Entradas:
    • Verifica si anio1 (primer año) es menor o igual a anio2 (segundo año).
    • Si anio1 > anio2, muestra un mensaje de error y continúa el bucle para solicitar nuevamente.
  • Bloque except:
    • Captura excepciones del tipo ValueError, que ocurren si el usuario ingresa algo que no puede convertirse a entero (por ejemplo, letras o símbolos).
    • Muestra un mensaje de error y continúa el bucle.
  • Retorno: Cuando las entradas son válidas y anio1 <= anio2, retorna una tupla (anio1, anio2).

Importancia de la Validación:

  • Prevención de Errores: Asegura que el programa no falle debido a entradas inválidas.
  • Experiencia del Usuario: Proporciona retroalimentación clara sobre lo que está mal y cómo corregirlo.
  • Integridad de los Datos: Garantiza que las operaciones subsiguientes (como el cálculo de años bisiestos) se realicen con datos correctos.

4. Función realizar_consulta()

def realizar_consulta():
    """Realiza una consulta de años bisiestos según la entrada del usuario."""
    anio_inicio, anio_fin = solicitar_años()
    bisiestos = obtener_años_bisiestos(anio_inicio, anio_fin)
    cantidad = len(bisiestos)
    
    print(f"\nEntre los años {anio_inicio} y {anio_fin} hay {cantidad} año(s) bisiesto(s).")
    if cantidad > 0:
        print("Años bisiestos encontrados:")
        print(", ".join(map(str, bisiestos)))
    print()

Propósito: Coordina el proceso de una única consulta para encontrar años bisiestos entre dos años proporcionados por el usuario.

Cómo Funciona:

  1. Obtención de las Entradas Validadas:
    • Llama a solicitar_años() para recibir los dos años.
  2. Cálculo de los Años Bisiestos:
    • Utiliza obtener_años_bisiestos(anio_inicio, anio_fin) para obtener la lista de años bisiestos en el intervalo.
  3. Conteo de los Años Bisiestos:
    • len(bisiestos): Calcula cuántos años bisiestos se encontraron.
  4. Exhibición de los Resultados:
    • Imprime un mensaje informando la cantidad de años bisiestos en el intervalo.
    • Si hay años bisiestos (cantidad > 0), lista esos años, separados por comas.
    • El uso de ", ".join(map(str, bisiestos)) convierte cada número de la lista a cadena y los une con comas.

Ejemplo de Salida:

Entre los años 2000 y 2020 hay 6 año(s) bisiesto(s).
Años bisiestos encontrados:
2000, 2004, 2008, 2012, 2016, 2020

5. Función main()

def main():
    """Función principal del programa."""
    print("Bienvenido al contador de años bisiestos.")
    while True:
        realizar_consulta()
        while True:
            respuesta = input("¿Desea realizar otra consulta? (s/n): ").strip().lower()
            if respuesta == 's':
                print()
                break
            elif respuesta == 'n':
                print("\nGracias por usar el programa. ¡Hasta luego!")
                return
            else:
                print("Respuesta inválida. Por favor, ingrese 's' para sí o 'n' para no.")

Propósito: Controla el flujo principal del programa, permitiendo que el usuario realice múltiples consultas o finalice el programa según lo desee.

Cómo Funciona:

  1. Mensaje de Bienvenida:
    • Imprime "Bienvenido al contador de años bisiestos."
  2. Bucle Principal (while True):
    • Permite que el usuario realice varias consultas sucesivamente.
  3. Llamada a realizar_consulta():
    • Ejecuta una consulta completa de años bisiestos.
  4. Bucle Interno para Preguntar si Desea Continuar:
    • Después de cada consulta, pregunta al usuario si desea realizar otra ("¿Desea realizar otra consulta? (s/n): ").
    • Entrada del Usuario:
      • .strip().lower(): Elimina espacios en blanco y convierte la respuesta a minúsculas para facilitar la comparación.
    • Condiciones:
      • Si la respuesta es 's' (sí):
        • Imprime una línea en blanco para separar las consultas.
        • Usa break para salir del bucle interno y comenzar una nueva consulta.
      • Si la respuesta es 'n' (no):
        • Imprime un mensaje de despedida y usa return para finalizar la función main(), terminando el programa.
      • De lo contrario:
        • Informa que la respuesta es inválida y solicita nuevamente.

Importancia de la Estructura:

  • Flujo Controlado: Asegura que el usuario pueda realizar múltiples consultas sin reiniciar el programa.
  • Validación de Respuesta: Evita comportamientos inesperados al manejar entradas inválidas.
  • Experiencia del Usuario: Proporciona una interacción amigable y clara.

6. Bloque de Ejecución

if __name__ == "__main__":
    main()

Propósito: Determina si el script está siendo ejecutado directamente o importado como módulo en otro script.

Cómo Funciona:

  • __name__:
    • En Python, cuando un script se ejecuta directamente, la variable especial __name__ se define como "__main__".
    • Si el script se importa como módulo en otro script, __name__ será el nombre del módulo.
  • Condición: if __name__ == "__main__": verifica si el script está siendo ejecutado directamente.
  • Acción:
    • Si es verdadero, llama a la función main() para iniciar el programa.
    • Esto permite que el código sea reutilizado como módulo sin ejecutar el programa principal automáticamente.

Importancia:

  • Reutilización de Código: Permite que las funciones del script sean usadas en otros scripts sin ejecutar el programa completo.
  • Organización: Mantiene la lógica de ejecución separada de la definición de funciones.
